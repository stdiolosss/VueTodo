# Project Name

VueTodo

## Description

This is a simple todo list app built with Vue.js and [json-server](https://github.com/typicode/json-server).

## Installation

```bash
npm install
npm install -g json-server
node db.js #generate db.json
```

## Running the Project

```bash
json-server db.json -p 3003
npm run serve
```

## Tech Stack

- Vue
- json-server
